## Jab hum Provisioning parhen ge to hum in technologies ke saath kaam karen ge:

1. VirtualBox
2. VMware Fusion
3. WSL (Windows Subsystem for Linux)
4. GCP (Google Cloud Platform)
5. AWS (Amazon Web Services)
6. Azure

Yeh tamam mukhtalif provisioning platforms hain jinhein hum course ke dauran seekhen ge.

### Note

Provisioning ka matlab hai servers, virtual machines, aur cloud resources ko create, configure aur manage karna. Is course mein hum local virtualization se lekar cloud environments tak mukhtalif platforms par Linux systems ko deploy aur manage karna seekhen ge.
